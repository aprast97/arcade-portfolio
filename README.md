# 8-Bit Synthwave Portfolio

A retro-styled portfolio website featuring an 8-bit aesthetic with a Synthwave color palette (Dark Purple, Cyan, Neon Pink).

## Features

- **Retro Aesthetic**: Pixelated fonts ('Press Start 2P'), scanline effects, and CRT monitor vibes.
- **Synthwave Theme**: Custom color palette inspired by Retrowave/Synthwave art.
- **Interactive Elements**:
  - 8-bit sound effects on button hover and click.
  - Animated text and borders.
  - "Insert Coin" start screen experience.
- **Responsive Design**: Works on desktop and mobile screens.

## Sections

1.  **Start Screen (Home)**: Main menu navigation.
2.  **Profile**: Character stats style user profile with social links (LinkedIn, GitHub).
3.  **Education**: Training log style education history.
4.  **Projects**: Mission log style project showcase with links.

## How to Run

Simply open `index.html` in any modern web browser.

## Technologies Used

- HTML5
- CSS3 (Animations, Flexbox, CSS Variables)
- JavaScript (Web Audio API for sound effects)
- Google Fonts (Press Start 2P)
